**Logs** `package documentation`
=================================================

Пакет, содержащий конфигурации для логирования работы клиентского и серверного модулей.

* Скрипт client_log.py содержит конфигурацию клиентского логгера.
* Скрипт server_log.py содержит конфигурацию серверного логгера.
